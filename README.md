# Dependencies

* **Node** >= *v15.0.1*

* **NPM**  >= *v7.0.3*
* **Rails** >= *v6.0.3.4*

# UI

* For project UI used Material-UI library. Read more [here](https://material-ui.com/ru/).

# Available commands

* ```make server```
    To run server on port 3001. Visist http://localhost.3001.

* ```make client```
    To run client app on port 3000. Visist http://localhost.3000.
